# Solution Elements

## Lösungsidee

LocalLend verbindet Personen, die Gegenstände verleihen möchten, mit Personen, die diese Gegenstände benötigen.

Die Plattform ermöglicht das lokale Verleihen und Ausleihen von Werkzeugen und technischen Geräten.

---

## Zentrale Erkenntnisse

Aus der Product Discovery wurden folgende Erkenntnisse abgeleitet:

1. Viele Nutzer benötigen Werkzeuge oder technische Geräte nur selten.
2. Viele Nutzer möchten lieber ausleihen als kaufen.
3. Vertrauen ist ein zentraler Erfolgsfaktor.
4. Verleiher möchten Kontrolle über ihre Gegenstände behalten.
5. Nutzer benötigen eine einfache Suche und klare Statusinformationen.

---

## Hauptfunktionen

| Funktion | Beschreibung |
|---|---|
| Registrierung | Nutzer können ein Konto erstellen |
| Login | Nutzer können sich anmelden |
| Gegenstände erstellen | Nutzer können eigene Gegenstände anbieten |
| Suche | Nutzer können verfügbare Gegenstände finden |
| Kategorien | Gegenstände können übersichtlich gruppiert werden |
| Detailansicht | Nutzer erhalten Informationen zu einem Gegenstand |
| Anfrage-System | Nutzer können Ausleihanfragen stellen |
| Statusverwaltung | Anfragen erhalten Statuswerte wie pending, accepted oder rejected |
| JSON-API | Ausgewählte Daten werden strukturiert bereitgestellt |

---

## Nutzerbedürfnis und Umsetzung

| Nutzerbedürfnis | Umsetzung in LocalLend |
|---|---|
| Gegenstände schnell finden | Suche und Übersicht |
| Orientierung behalten | Kategorien |
| Gegenstände einfach anfragen | Anfrage-System |
| Kontrolle behalten | Anfragen annehmen oder ablehnen |
| Transparenz schaffen | Statusverwaltung |
| Vertrauen unterstützen | Login und manuelle Bestätigung |

---

## Priorisierung

| Priorität | Funktion | Begründung |
|---|---|---|
| Hoch | Anfrage-System | Kernfunktion der Plattform |
| Hoch | Statusverwaltung | Macht den Anfrageprozess nachvollziehbar |
| Hoch | Suche | Nutzer möchten Gegenstände schnell finden |
| Hoch | Login | Unterstützt Vertrauen und Nachvollziehbarkeit |
| Mittel | Kategorien | Verbessert die Orientierung |
| Mittel | Detailansicht | Liefert wichtige Informationen |
| Mittel | JSON-API | Erfüllt technische Projektanforderungen |
